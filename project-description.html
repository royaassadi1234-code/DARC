Project description

A digital workspace for reading and comparing Iranian religious texts.

This project brings together searchable passages, translations, comments, themes, and comparison tools for Avestan and Middle Persian materials.

Its main aim is to make textual patterns easier to see: where words repeat, where concepts change meaning, and how religious ideas move across related sources.
